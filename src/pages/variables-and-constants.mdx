# Переменные и Константы в JavaScript

В JavaScript есть два основных способа объявления переменных: с помощью `let` и `const`.

## Переменные (`let`)

Переменные, объявленные с помощью `let`, могут изменять свое значение после инициализации.

```javascript
let myVariable = 'Начальное значение';
myVariable = 'Новое значение';
console.log(myVariable); // Выведет 'Новое значение'
```

̌Константы (const)
Константы, объявленные с помощью const, не могут изменять свое значение после инициализации.

```javascript
const MY_CONSTANT = 'Начальное значение';
// Попытка изменить значение приведет к ошибке
// MY_CONSTANT = 'Новое значение'; // Uncaught TypeError: Assignment to constant variable.
```
