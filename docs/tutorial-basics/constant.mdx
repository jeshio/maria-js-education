# 1.1 Константы

## Знания

:::tip[Что это такое?]
Константа содержит значение (число, строку или другой __тип данных__), которое в дальнейшем, по коду, нельзя изменить.
:::

В JavaScript для объявления константы используется __ключевое слово__ `const`.

Пример создания константы (научным языком это называется __инициализация__ или __объявление__)

```javascript
const superConst = 'Начальное значение';
// А это комментарий в коде, после двух слешей может идти любой поясняющий текст
// Этот текст для тебя, разработчика, JS его игнорирует

superConst = 'Новое значение'; // <-- Константу нельзя изменять после инициализации
// JS будет ругаться и выдаст ошибку с соответствующим текстом на английском языке
```

`superConst` в этом примере - это название константы, а с помощью знака равна этой константе присваивается значение, которое потом можно использовать.

Каждая строка, которую JS будет исполнять (то есть, всё, кроме комментариев) заканчивается знаком `;`. Грубо говоря, мы таким образом разделяем команды, которые даём компьютеру.

:::tip
С помощью знака `=` мы говорим компьютеру, чтобы константа хранила значение, которое мы пишем после этого знака.
:::

В нашем примере с помощью знака `=` мы говорим компьютеру, что хотим __задать значение__ для константы.

Соответственно для компьютера строка `const superConst = 'Начальное значение';` звучит так:

```
Объяви константу с названием superConst, и пусть её значение будет 'Начальное значение'
```

Это первая команда, которую мы даем компьютеру. А вторая команда - `superConst = 'Новое значение';`. Тут мы говорим, чтобы компьютер присвоил константе новое значение.
Но компьютер ответит, что константе нельзя устанавливать новое значение. Она может хранить только изначально установленное.

:::danger[_Константа_ в переводе с латинского _Постоянная_]
После иницлизации константы, её уже нельзя изменить. JS выдаст ошибку `Assignment to constant variable.`
:::

Также, обрати внимание, что `const` используется для __объявления__ константы. Но если ты её не объявишь, а попробуешь сделать так:

```javascript
superConst = 'Начальное значение'; // слово const в начале не написали
```

То компьютер ответит, что константы superConst не существует. А всё потому, что ты не объявила ему, что она существует.

:::tip[Кто исполняет наши команды?]
В общем говоря наши команды исполняет процессор в компьютере или телефоне, на котором запускается наш код.

Но, если говорить более конкретно, то в случае когда код запускается на сайте, как здесь, команды исполняет браузер.
:::
