# 1.1 Введение в HTML и VS Code

## Зачем нужен HTML?

:::tip[Что такое HTML?]
__HTML__ (HyperText Markup Language) — это язык разметки, который используется для создания структуры веб-страниц. Он определяет, какие элементы будут отображаться на странице: текст, изображения, ссылки, таблицы и многое другое.
:::

Представь HTML как "скелет" веб-страницы. Он определяет, где заголовок, где текст, где картинка, а потом к этому добавляются стили (_CSS_) и функциональность (_JavaScript_).

Пример HTML-кода:
```html
<!DOCTYPE html>
<html>
  <head>
    <title>Моя первая страница</title>
  </head>
  <body>
    <h1>Привет, мир!</h1>
    <p>Это мой первый веб-сайт.</p>
  </body>
</html>
```

## Установка и настройка VS Code

Для написания HTML-кода мы будем использовать текстовый редактор [VS Code](https://code.visualstudio.com/). Это удобный и популярный инструмент для разработчиков.

### Установка

1. Перейди на сайт [VS Code](https://code.visualstudio.com/) и скачай версию для своей операционной системы.
2. Установи VS Code, следуя инструкциям на экране.

### Настройка для удобной работы

1. Открой VS Code.
2. Установи расширение **"Live Server"**:
- Нажми на иконку Extensions (слева, в виде квадратиков).
- Введи "Live Server" в поиске и нажми "Install".
3. Теперь у тебя есть инструмент для быстрого просмотра HTML-страниц в браузере.

## Создание первой HTML-страницы

### Шаги:

1. Придумай, где в какой папке ты будешь хранить свои начальные проекты
2. Внутри этой папки создай новую `1-html-basic`
3. Открой эту папку в VS Code (`File -> Open folder`)
4. Слева в VS Code будут отображаться файлы внутри твоей папки. Нажми правой кнопокй и создай новый файл с названием `index.html`.
5. Вставь следующий код:

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Моя первая страница</title>
  </head>
  <body>
    <h1>Привет, мир!</h1>
    <p>Это мой первый веб-сайт.</p>
  </body>
</html>
```

6. Сохрани файл.

---

## Просмотр страницы в браузере

Попробуй сначала первй способ просомтра файла, а затем второй. Выбери, который тебе больше нравиться.

### Способ 1: Открыть напрямую
1. Перейди в папку, где находится файл `index.html`.
2. Дважды щёлкни по файлу, чтобы открыть его в браузере.
3. Если ты что-то изменишь и сохранишь файл, то для просмотра изменений, в браузере нужно обновить страницу (можно использовать `Command+R`).

### Способ 2: Через Live Server
1. В VS Code щёлкни правой кнопкой мыши на файле `index.html`.
2. Выбери "Open with Live Server".
3. Страница откроется в браузере, и любые изменения в файле будут отображаться мгновенно.

## Структура HTML-документа

Давай разберём код, который ты ввела:

1. `<!DOCTYPE html>` — сообщает браузеру, что это документ HTML5. Просто стандартное начало.
2. `<html>` — корневой элемент документа, с которого мы начинаем спускаться вниз.
3. `<head>` — содержит метаинформацию (заголовок страницы, кодировку, стили и т.д.). Эту информацию пользователь не увидит в браузере, за исключением тега `title`, который отображается в названии вкладки в браузере. Также, здесь размещаются всякие теги для поисковиков.
4. `<title>` — название страницы, которое отображается на вкладке браузера.
5. `<body>` — это основной тег, внутри которого будет почти весь твой HTML код. Всё, что внутри `<body>`, отображается на странице:
- `<h1>` — заголовок.
- `<p>` — абзац текста.

1. Открой `index.html` и измени заголовок (`<h1>`) на своё имя.

3. Проверь изменения в браузере.
