# 1.2 Изображения, ссылки и стили текста

## Добавление изображений

:::tip[Тег `<img>`]
Для вставки изображения используется тег `<img>`. Он:
- __Не имеет закрывающего тега__, т.е. пишется в одну строку.
- Обязательно содержит атрибут `src`, который указывает путь к файлу изображения.
- Может содержать атрибут `alt` для альтернативного текста (важно для доступности людям с ограниченными возможностями).

Пример:
```html
<img src="example.jpg" alt="Пример изображения">
```
:::

### Пример:

```html
<img src="https://via.placeholder.com/300" alt="Заглушка изображения">
```

На странице отобразится картинка размером 300x300 пикселей.

## Добавление ссылок

### Внешние ссылки

:::tip[Тег `<a>`]
Для создания ссылки используется тег `<a>`. Он:
- Обязательно содержит атрибут `href`, указывающий адрес, куда ведёт ссылка.
- Может содержать текст или другие элементы, которые будут кликабельными.

Пример:
```html
<a href="https://google.com">Перейти в Google</a>
```
:::

### Пример:

```html
<a href="https://example.com" target="_blank">Мой любимый сайт</a>
```

Здесь:
- `target="_blank"` открывает ссылку в новой вкладке.

---

### Локальные ссылки

Локальные ссылки ведут на другие файлы внутри твоего проекта. Например, если у тебя есть файл `about.html` в той же папке, что и `index.html`, ты можешь сделать ссылку на него:

```html
<a href="about.html">Открыть страницу "Обо мне"</a>
```

Чтобы создать другую страницу, просто добавь новый HTML-файл в ту же папку и добавь локальную ссылку на этот файл.

---

## Выделение текста жирным шрифтом

:::tip[Теги `<b>` и `<strong>`]
- `<b>` используется для чисто визуального выделения текста жирным шрифтом.
- `<strong>` делает текст жирным и добавляет семантический акцент, что важно для поисковых систем и доступности.

Пример:
```html
<p>Это <b>жирный текст</b>, а это <strong>важный текст</strong>.</p>
```
:::

## Пример: объединим всё вместе

Добавим на страницу изображение, ссылку, локальную ссылку и немного текста с разными стилями.

```html
<!DOCTYPE html>
<html>
  <head>
    <title>Изображения и ссылки</title>
  </head>
  <body>
    <h1>Добро пожаловать на мою страницу</h1>
    <p>Это <b>жирный текст</b>, а вот <strong>важная информация</strong>.</p>
    <img src="https://via.placeholder.com/400" alt="Пример изображения">
    <p>Посетите <a href="https://example.com" target="_blank">мой любимый сайт</a>!</p>
    <p><a href="about.html">Перейти на страницу "Обо мне"</a></p>
  </body>
</html>
```

## Практическое задание

1. Добавь на свою страницу изображение с URL: `https://via.placeholder.com/600x200` и подпись под ним: "Это мой баннер".
2. Создай локальную страницу `contact.html` с заголовком "Контакты" и текстом "Напишите нам!".
3. Добавь ссылку в `index.html`, которая ведёт на страницу "Контакты".
4. На странице `contact.html` добавь ссылку "Вернуться на главную", которая ведёт на `index.html`.
5. Напиши абзац текста, где выдели одну часть жирным с помощью `<b>`, а другую — с помощью `<strong>`.

<details>
    <summary>Подсказка</summary>

    ### index.html
    ```html
    <!DOCTYPE html>
    <html>
    <head>
        <title>Главная страница</title>
    </head>
    <body>
    <h1>Добро пожаловать!</h1>
    <img src="https://via.placeholder.com/600x200" alt="Это мой баннер">
        <p>Это <b>жирный текст</b>, а вот <strong>важное сообщение</strong>.</p>
        <p>Посетите <a href="contact.html">страницу "Контакты"</a>.</p>
    </body>
    </html>
    ```

    ```html
    <!DOCTYPE html>
    <html>
    <head>
        <title>Контакты</title>

    <h1>Контакты</h1>

    <p><a href="index.html">Вернуться на главную</a></p>
    </body>
    </html>
    ```
