# 1.3 Списки, таблицы и простой JavaScript

## Введение

На этом уроке мы научимся создавать списки и таблицы, чтобы лучше организовать контент на странице. А ещё, добавим простой JavaScript-код.

---

## Списки

### Нумерованные списки

Для создания нумерованного списка используется тег `<ol>` (от *ordered list* — "упорядоченный список"). Каждый элемент списка создаётся с помощью тега `<li>`.

Пример:
```html
<ol>
  <li>Первый пункт</li>
  <li>Второй пункт</li>
  <li>Третий пункт</li>
</ol>
```

Результат:
1. Первый пункт
2. Второй пункт
3. Третий пункт

---

### Маркированные списки

Для создания маркированного списка используется тег `<ul>` (от *unordered list* — "неупорядоченный список").

Пример:
```html
<ul>
  <li>Красный</li>
  <li>Синий</li>
  <li>Зелёный</li>
</ul>
```

Результат:
- Красный
- Синий
- Зелёный

---

## Таблицы

:::tip[Теги для таблиц]
- `<table>` — создаёт таблицу.
- `<tr>` (table row) — строка таблицы.
- `<td>` (table data) — ячейка в строке.
- `<th>` (table header) — заголовок колонки в таблице, который выделяется жирным шрифтом. Обычно используется только в первой строке.
:::

Пример:
```html
<table border="1">
  <tr>
    <th>Имя</th>
    <th>Возраст</th>
  </tr>
  <tr>
    <td>Анна</td>
    <td>25</td>
  </tr>
  <tr>
    <td>Олег</td>
    <td>30</td>
  </tr>
</table>
```

Результат:

| Имя  | Возраст |
|------|---------|
| Анна | 25      |
| Олег | 30      |

## Добавляем JavaScript

### Первый скрипт: работа с текстом

Скрипты добавляются с помощью тега `<script>`. Обычно этот тег помещается в самый низ тега `body`.

Пример:
```html
<!DOCTYPE html>
<html>
  <head>
    <title>Заголовок вкладки</title>
  </head>
  <body>
    <h1>Контент будет здесь</h1>
    <p>Это мой веб-сайт. Тут даже скрипт есть!</p>
    <script>
      console.log('Привет, мир!');
    </script>
  </body>
</html>
```

---

### Взаимодействие с HTML

JavaScript может изменять содержимое страницы. Например, с помощью метода `document.getElementById` мы можем найти элемент по его `id` и изменить текст внутри.

Пример:
```html
<!DOCTYPE html>
<html>
  <head>
    <title>JavaScript</title>
  </head>
  <body>
    <h1 id="greeting">Привет!</h1>
    <button onclick="changeGreeting()">Нажми меня</button>

    <script>
      function changeGreeting() {
        const greeting = document.getElementById('greeting');
        greeting.textContent = 'Привет, мир!';
      }
    </script>
  </body>
</html>
```

Здесь:
- `id="greeting"` указывает идентификатор элемента.
- `onclick="changeGreeting()"` привязывает событие к кнопке. То есть, при нажатии кнопки, будет вызываться функция `changeGreeting`
- функция `changeGreeting` меняет текст внутри элемента `<h1>`. Для этого мы получаем элемент `<h1>` по аттрибуту `id` - JavaScript будет видеть его объектом с разными свойствами и методами. Свойство `textContent` отвечает за текст внутри элемента. Мы его можем извлечь, либо изменить.

## Практическое задание

1. Создай html страницу
2. Добавь заголовок `h1` с текстом `Привет!`
3. Создай нумерованный список с твоими любимыми фильмами.
4. Создай таблицу, где каждая строка описывает продукт:
- Название.
- Цена.
- Количество.
5. Добавь в таблицу несколько продкутов из Ленты.
6. Добавь кнопку с текстом `Я всё`, которая при нажатии изменяет текст в `<h1>` на `Пока!`.
