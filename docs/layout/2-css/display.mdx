# 2.7 –°–≤–æ–π—Å—Ç–≤–æ display –∏ —Ç–∏–ø—ã —ç–ª–µ–º–µ–Ω—Ç–æ–≤

–í —ç—Ç–æ–º —É—Ä–æ–∫–µ —è —Ä–∞—Å—Å–∫–∞–∂—É —Ç–µ–±–µ, –∫–∞–∫ —Ä–∞–±–æ—Ç–∞–µ—Ç —Å–≤–æ–π—Å—Ç–≤–æ display, –∫–∞–∫–∏–µ –±—ã–≤–∞—é—Ç —Ç–∏–ø—ã —ç–ª–µ–º–µ–Ω—Ç–æ–≤ (–±–ª–æ—á–Ω—ã–µ, —Å—Ç—Ä–æ—á–Ω—ã–µ –∏ –¥—Ä.), –∏ –∫–∞–∫ —ç—Ç–æ –≤–ª–∏—è–µ—Ç –Ω–∞ –ø–æ—Å—Ç—Ä–æ–µ–Ω–∏–µ –º–∞–∫–µ—Ç–∞ —Å—Ç—Ä–∞–Ω–∏—Ü—ã.

## –ß—Ç–æ —Ç–∞–∫–æ–µ display

–°–≤–æ–π—Å—Ç–≤–æ `display` –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç, –∫–∞–∫ —ç–ª–µ–º–µ–Ω—Ç –±—É–¥–µ—Ç –≤–µ—Å—Ç–∏ —Å–µ–±—è –Ω–∞ —Å—Ç—Ä–∞–Ω–∏—Ü–µ ‚Äî –±—É–¥–µ—Ç –ª–∏ –æ–Ω –∑–∞–Ω–∏–º–∞—Ç—å –≤—Å—é —Å—Ç—Ä–æ–∫—É, —Ä–∞—Å–ø–æ–ª–∞–≥–∞—Ç—å—Å—è –≤ –æ–¥–Ω—É –ª–∏–Ω–∏—é —Å –¥—Ä—É–≥–∏–º–∏, –∏–ª–∏ –≤–æ–æ–±—â–µ –Ω–µ –æ—Ç–æ–±—Ä–∞–∂–∞—Ç—å—Å—è.

## –û—Å–Ω–æ–≤–Ω—ã–µ –∑–Ω–∞—á–µ–Ω–∏—è display

- `block` ‚Äî –±–ª–æ—á–Ω—ã–π —ç–ª–µ–º–µ–Ω—Ç (–∑–∞–Ω–∏–º–∞–µ—Ç –≤—Å—é —à–∏—Ä–∏–Ω—É, –Ω–∞—á–∏–Ω–∞–µ—Ç—Å—è —Å –Ω–æ–≤–æ–π —Å—Ç—Ä–æ–∫–∏)
- `inline` ‚Äî —Å—Ç—Ä–æ—á–Ω—ã–π —ç–ª–µ–º–µ–Ω—Ç (—Ä–∞—Å–ø–æ–ª–∞–≥–∞–µ—Ç—Å—è –≤ –æ–¥–Ω—É –ª–∏–Ω–∏—é —Å –¥—Ä—É–≥–∏–º–∏, —à–∏—Ä–∏–Ω–∞ –ø–æ —Å–æ–¥–µ—Ä–∂–∏–º–æ–º—É)
- `inline-block` ‚Äî —Å—Ç—Ä–æ—á–Ω–æ-–±–ª–æ—á–Ω—ã–π (–∫–∞–∫ —Å—Ç—Ä–æ—á–Ω—ã–π, –Ω–æ –º–æ–∂–Ω–æ –∑–∞–¥–∞–≤–∞—Ç—å —à–∏—Ä–∏–Ω—É –∏ –≤—ã—Å–æ—Ç—É)

## –ë–ª–æ—á–Ω—ã–µ –∏ —Å—Ç—Ä–æ—á–Ω—ã–µ —ç–ª–µ–º–µ–Ω—Ç—ã

- –ë–ª–æ—á–Ω—ã–µ: `<div>`, `<p>`, `<h1>`, `<ul>`, `<li>`, `<section>`, `<header>` –∏ –¥—Ä.
  - –ë–ª–æ—á–Ω—ã–µ —ç–ª–µ–º–µ–Ω—Ç—ã –≤—Å–µ–≥–¥–∞ –∑–∞–Ω–∏–º–∞—é—Ç –≤—Å—é —à–∏—Ä–∏–Ω—É —Ä–æ–¥–∏—Ç–µ–ª—è, –¥–∞–∂–µ –µ—Å–ª–∏ —Å–æ–¥–µ—Ä–∂–∏–º–æ–≥–æ –º–∞–ª–æ. –û–Ω–∏ –Ω–∞—á–∏–Ω–∞—é—Ç—Å—è —Å –Ω–æ–≤–æ–π —Å—Ç—Ä–æ–∫–∏ –∏ "–≤—ã–¥–∞–≤–ª–∏–≤–∞—é—Ç" —Å–ª–µ–¥—É—é—â–∏–µ —ç–ª–µ–º–µ–Ω—Ç—ã –≤–Ω–∏–∑.
- –°—Ç—Ä–æ—á–Ω—ã–µ: `<span>`, `<a>`, `<strong>`, `<em>`, `<img>` (–ø–æ —É–º–æ–ª—á–∞–Ω–∏—é), `<input>`, `<label>`
  - –°—Ç—Ä–æ—á–Ω—ã–µ —ç–ª–µ–º–µ–Ω—Ç—ã —Ä–∞—Å–ø–æ–ª–∞–≥–∞—é—Ç—Å—è –≤ –æ–¥–Ω—É –ª–∏–Ω–∏—é —Å –¥—Ä—É–≥–∏–º–∏, —à–∏—Ä–∏–Ω–∞ –∏ –≤—ã—Å–æ—Ç–∞ —É –Ω–∏—Ö –∑–∞–≤–∏—Å—è—Ç —Ç–æ–ª—å–∫–æ –æ—Ç —Å–æ–¥–µ—Ä–∂–∏–º–æ–≥–æ. –î–ª—è –Ω–∏—Ö –Ω–µ–ª—å–∑—è –∑–∞–¥–∞—Ç—å width/height, –∞ –≤–µ—Ä—Ç–∏–∫–∞–ª—å–Ω—ã–µ margin –∏ padding —á–∞—Å—Ç–æ –Ω–µ —Ä–∞–±–æ—Ç–∞—é—Ç –∫–∞–∫ –æ–∂–∏–¥–∞–µ—Ç—Å—è (–∏—Å–ø–æ–ª—å–∑—É–π –≥–æ—Ä–∏–∑–æ–Ω—Ç–∞–ª—å–Ω—ã–µ).
