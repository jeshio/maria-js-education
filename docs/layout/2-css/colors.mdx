# 2.5 Цвета и фоны в CSS

В этом уроке я расскажу тебе, как управлять цветами и фоновыми свойствами в CSS, а ещё покажу важный приём работы с селекторами, когда у элемента несколько классов.

## Цветовые значения в CSS

В CSS ты можешь задавать цвет разными способами:

- **Ключевые слова**: `red`, `blue`, `green`, `black`, `white` и другие.
- **HEX-код**: `#ff0000`, `#00ff00`, `#0000ff`
- **RGB**: `rgb(255, 0, 0)`
- **RGBA** (с прозрачностью): `rgba(255, 0, 0, 0.5)`
- **HSL**: `hsl(120, 100%, 50%)`
- **HSLA**: `hsla(120, 100%, 50%, 0.3)`

```css
.color-demo {
  color: #ff6600;
  background-color: rgb(240, 240, 240);
}
```

:::tip[Когда использовать]
- HEX — подойдёт для большинства задач
- RGB/RGBA — используй, если нужна прозрачность
- HSL/HSLA — удобно, если хочешь динамически менять цвет
:::

## Фоновые свойства

С помощью CSS ты можешь задавать не только цвет фона, но и использовать изображения, градиенты и даже несколько фонов одновременно.

#### background-color, background-image и background

- `background-color` — задаёт только цвет фона.
- `background-image` — только фоновое изображение.
- `background` — это сокращённое свойство, с помощью которого ты можешь задать сразу несколько параметров: цвет, изображение, позицию, повторение, размер и так далее.

```css
.box1 {
  background-color: #e0f7fa; /* только цвет */
}

.box2 {
  background-image: url('pattern.png'); /* только картинка */
}

.box3 {
  background: #e0f7fa url('pattern.png') right top/50px 50px no-repeat; /* всё вместе */
}
```

**Когда что использовать:**
- Если нужен только цвет — используй `background-color`.
- Если только картинка — `background-image`.
- Если хочешь задать сразу несколько параметров (цвет, картинку, позицию, повторение и т.д.) — используй сокращённое свойство `background`.
- Если ты используешь `background`, оно перезапишет все отдельные фоновые свойства, которые были заданы раньше для этого элемента.

:::tip[Совет]
Для простоты и читаемости кода часто удобно использовать отдельные свойства, а для сложных фонов — сокращённую запись.
:::

### Фоновое изображение

```css
.header {
  background-image: url('header-bg.jpg');
  background-repeat: no-repeat;
  background-size: cover;
}
```

### Градиенты

В CSS ты можешь использовать линейные и радиальные градиенты:

```css
.gradient {
  background: linear-gradient(90deg, #ff8a00, #e52e71);
}

.circle-gradient {
  background: radial-gradient(circle, #fff 0%, #2196f3 100%);
}
```

### Несколько фонов

Ты можешь комбинировать изображения, градиенты и цвета:

```css
.multi-bg {
  background:
    url('pattern.png') repeat,
    radial-gradient(circle, #fff 0%, #2196f3 100%),
    #e0f7fa;
}
```

:::tip
Порядок перечисления фонов важен: __первый фон будет самым верхним слоем__, а __последний — самым нижним (базовым)__. Например, цвет обычно указывают последним, чтобы он был "подложкой" для градиентов и картинок.

Используй это, чтобы создавать сложные и красивые эффекты наложения!
:::

## Использование нескольких классов и селектор `.class1.class2`

В CSS элемент может иметь сразу несколько классов через пробел:

```html
<div class="card highlight">...</div>
```

Если ты хочешь применить стиль только к элементу, у которого есть оба класса, используй селектор `.card.highlight` (без пробела!):

```css
.card.highlight {
  box-shadow: 0 4px 16px rgba(0,0,0,0.15);
  background-color: #fffbe6;
}
```

- `.card .highlight` (с пробелом) — выберет элемент с классом `highlight` внутри элемента с классом `card`.
- `.card.highlight` (без пробела) — выберет элемент, у которого одновременно есть оба класса.

:::info[Практика]
Если ты хочешь, чтобы какой-то элемент выглядел по-особенному (например, выделить акционный товар), просто добавь ему второй класс и опиши нужный стиль через селектор с двумя классами. Это очень удобно: не нужно придумывать новые классы для каждого случая — ты просто комбинируешь уже существующие!

Например, если у тебя есть обычная карточка товара и ты хочешь выделить акционный вариант, добавь класс highlight:

```html
<div class="product-card highlight">...</div>
```

И опиши стиль для .product-card.highlight — только такие карточки будут выделяться.
:::

### Влияние на приоритет (специфичность)

Когда ты используешь селектор с несколькими классами, их "очки" складываются. Каждый класс даёт 10 очков специфичности:

- `.card` — 10 очков
- `.highlight` — 10 очков
- `.card.highlight` — 20 очков (10 + 10)

Это значит, что правило с двумя классами будет иметь больший приоритет, чем правило с одним классом, но меньше, чем правило с ID:

```css
.card {
  background: #f5f5f5; /* 10 очков */
}
.card.highlight {
  background: #fffbe6; /* 20 очков — перекроет .card */
}
#main {
  background: #e0f7fa; /* 100 очков — перекроет оба класса */
}
```

```html
<div class="card highlight" id="main">...</div>
```

В этом примере фоновый цвет будет `#e0f7fa`, потому что селектор с ID имеет больший приоритет. Если убрать id, применится стиль `.card.highlight`.

:::tip[Запомни]
Чем больше классов в селекторе, тем выше его специфичность (приоритет). Но не стоит этим злоупотреблять — сложные селекторы усложняют поддержку кода.
:::

## Практическое задание

Попробуй создать карточку товара с разными вариантами оформления:

1. Создай файл `product.html`:

```html
<!DOCTYPE html>
<html>
<head>
  <title>Карточка товара</title>
  <link rel="stylesheet" href="product.css">
</head>
<body>
  <div class="product-card">
    <h2 class="product-title">Смартфон X</h2>
    <div class="product-image"></div>
    <p class="product-desc">Современный смартфон с отличной камерой.</p>
    <div class="product-price highlight">29 990 ₽</div>
  </div>
  <div class="product-card highlight">
    <h2 class="product-title">Смартфон Y</h2>
    <div class="product-image"></div>
    <p class="product-desc">Мощный аккумулятор и стильный дизайн.</p>
    <div class="product-price">34 990 ₽</div>
  </div>
</body>
</html>
```

2. Создай файл `product.css` и добавь стили:
- Используй разные способы задания цвета
- Применяй фоны, градиенты, изображения
- Используй селектор с двумя классами для выделения акций
