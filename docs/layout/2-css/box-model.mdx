# 2.6 Блочная модель и отступы

В этом уроке я расскажу тебе, как устроена блочная модель в CSS, что такое margin, padding, border, как задавать размеры элементов и почему важно свойство box-sizing.

## Что такое блочная модель

Каждый элемент на странице можно представить как коробку (box), у которой есть:

- **Содержимое** (content) — сам текст или картинка внутри элемента
- **Внутренний отступ** (padding) — пространство между содержимым и границей
- **Граница** (border) — рамка вокруг элемента
- **Внешний отступ** (margin) — пространство между этим элементом и другими

Вот как это выглядит:

![Блочная модель](/img/block-model.png)

## Свойства отступов и границ

- `margin` — внешний отступ (снаружи)
- `padding` — внутренний отступ (внутри)
- `border` — граница (рамка)

```css
.box {
  margin: 20px;
  padding: 16px;
  border: 2px solid #2196f3;
}
```

Ты можешь задавать отступы и границы по отдельным сторонам:

```css
.box {
  margin-top: 10px;
  margin-bottom: 20px;
  padding-left: 30px;
  border-right: 4px dashed #ff8a00;
}
```

## Схлопывание внешних отступов (margin collapsing)

Когда два вертикальных margin (например, у соседних блоков или у родителя и первого/последнего потомка) встречаются, они не складываются, а "схлопываются" — остаётся только больший из них.

Например:

```css
.block {
  margin-bottom: 40px;
}
.block2 {
  margin-top: 30px;
}
```

Если поставить эти блоки друг за другом, расстояние между ними будет не 70px, а 40px (больший из двух margin).

Это важно учитывать при верстке, чтобы не удивляться "пропавшим" или слишком большим отступам!

## Размеры элементов

- `width` — ширина содержимого
- `height` — высота содержимого
- `min-width`, `max-width`, `min-height`, `max-height` — минимальные и максимальные размеры

```css
.box {
  width: 300px;
  min-height: 100px;
}
```

## Как считается итоговый размер

По умолчанию итоговая ширина элемента складывается так:

```
width + padding (слева и справа) + border (слева и справа)
```

То есть, если ты напишешь:

```css
.box {
  width: 200px;
  padding: 20px;
  border: 2px solid #2196f3;
}
```

То реальная ширина будет: 200 + 20*2 + 2*2 = 244px

## Свойство box-sizing

Чтобы проще управлять размерами, используй свойство `box-sizing: border-box;` — тогда padding и border будут входить в указанную ширину и высоту:

```css
.box {
  width: 200px;
  padding: 20px;
  border: 2px solid #2196f3;
  box-sizing: border-box;
}
```

Теперь итоговая ширина будет ровно 200px!

:::tip
Рекомендую всегда использовать `box-sizing: border-box;` для всех элементов — так проще верстать и не запутаться с размерами.

Для этого достаточно написать один раз в каком-нибудь файле (обычно отдельный файл для глобальных стилей):

```css
* {
  box-sizing: border-box;
}
```
:::

## Практическое задание

Попробуй создать несколько блоков с разными отступами и размерами, чтобы увидеть, как они взаимодействуют друг с другом и как работает схлопывание margin:

1. Создай файл `box.html`:

```html
<!DOCTYPE html>
<html>
<head>
  <title>Блочная модель</title>
  <link rel="stylesheet" href="box.css">
</head>
<body>
  <div class="container">
    <div class="box box1">
      <h2>Блок 1</h2>
      <p>Первый блок с большими внешними отступами.</p>
    </div>
    <div class="box box2">
      <h2>Блок 2</h2>
      <p>Второй блок с другими отступами.</p>
    </div>
    <div class="box box3">
      <h2>Блок 3</h2>
      <p>Третий блок с внутренними отступами и рамкой.</p>
    </div>
  </div>
</body>
</html>
```

2. Создай файл `box.css` и добавь стили с разными margin, padding и border для каждого блока. Обрати внимание, как margin между блоками схлопываются!

```css

  padding: 40px 32px;
}

  margin-top: 0;
  margin-bottom: 16px;
  color: #2196f3;
}
```
