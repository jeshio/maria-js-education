# 2.2 Селекторы CSS

В прошлом уроке мы познакомились с базовыми селекторами CSS. Теперь изучим более сложные способы выбора элементов и их комбинирование.

## Комбинаторы селекторов

Комбинаторы позволяют создавать более точные правила для выбора элементов:

### Потомки (уходим в глубину через пробел)

Выбирает все элементы внутри другого элемента, независимо от уровня вложенности:

```css
/* Все параграфы внутри любого div на странице будут с синим текстом */
div p {
  color: blue;
}
```

```html
<div>
  <p>Этот текст будет синим</p>
  <section>
    <p>Этот тоже будет синим</p>
  </section>
</div>
<p>
    А этот останется чёрным
    <div>
        <p>Этот будет синим</p>
    </div>
</p>
```

### Дочерние элементы (>)

Выбирает только __прямых потомков__ (первый уровень вложенности):

```css
/* Только прямые потомки-параграфы внутри div */
div > p {
  color: red;
}
```

```html
<div>
  <p>Этот текст будет красным</p>
  <section>
    <p>А этот останется чёрным</p>
  </section>
</div>
```

### Соседние элементы (+)

Выбирает элемент, который идёт сразу после указанного:

```css
/* Параграф сразу после заголовка */
h1 + p {
  font-size: 20px;
}
```

```html
<h1>Заголовок</h1>
<p>Этот текст будет увеличен</p>
<p>А этот останется обычным</p>
```

## Селекторы атрибутов

Позволяют выбирать элементы по их атрибутам. Это очень удобно при работе с формами и разными элементами.

### Точное совпадение

Примеры для общего понимания:

```css
/* Элементы с определённым атрибутом */
input[required] {
  border-color: red;
}

/* Элементы с определённым значением атрибута */
input[type="email"] {
  background-color: #f8f9fa;
}

/* Элементы с определённым классом (ещё один способ селектора по классу) */
div[class="important"] {
  font-weight: bold;
}

/* Кнопки с типом submit */
button[type="submit"] {
  background-color: #007bff;
  color: white;
}
```

### Частичное совпадение

```css
/* Содержит слово в списке классов. Например, элементы с классом primary-button, text-primary */
[class~="primary"] {
  color: blue;
}

/* Начинается с определённого значения. Например, кнопки с классом btn-primary и btn-secondary */
[class^="btn-"] {
  padding: 10px;
}

/* Заканчивается определённым значением. Например, ссылки на PDF-файлы */
[href$=".pdf"] {
  color: red;
}

/* Содержит определённое значение. Например, изображения в папке images */
[src*="images"] {
  border: 1px solid gray;
}
```

:::tip[Когда использовать]
Селекторы атрибутов особенно полезны когда:
- Работаешь с формами (type, required, disabled)
- Стилизуешь ссылки по типу файлов (pdf, doc)
- Нужно выбрать элементы с определёнными data-атрибутами
:::

## Группировка селекторов

Если нужно применить одинаковые стили к разным элементам, их можно сгруппировать через запятую:

```css
h1, h2, .important {
  color: purple;
}
```

```html
<h1>Заголовок фиолетового цвета</h1>
<h2>Подзаголовок фиолетового цвета</h2>
<p class="important">Текст фиолетового цвета</p>
```

## Практическое задание

Создай навигационное меню с вложенными списками и внешними ссылками:

1. Создай файл `menu.html`:
```html
<!DOCTYPE html>
<html>
<head>
  <title>Навигационное меню</title>
  <link rel="stylesheet" href="menu.css">
</head>
<body>
  <nav class="main-menu">
    <ul>
      <li>
        <a href="/" class="nav-link">Главная</a>
      </li>
      <li>
        <a href="/catalog" class="nav-link">Каталог</a>
        <ul>
          <li><a href="/catalog/electronics" class="nav-link">Электроника</a></li>
          <li><a href="/catalog/clothes" class="nav-link">Одежда</a></li>
          <li><a href="/files/catalog.pdf" class="nav-link" target="_blank">Скачать каталог</a></li>
        </ul>
      </li>
      <li>
        <a href="/about" class="nav-link">О нас</a>
        <ul>
          <li><a href="/about/history" class="nav-link">История</a></li>
          <li><a href="/about/contacts" class="nav-link" data-type="contact">Контакты</a></li>
          <li><a href="https://example.com" class="nav-link" target="_blank">Внешняя ссылка</a></li>
        </ul>
      </li>
    </ul>
  </nav>
</body>
</html>
```

2. Создай файл `menu.css` и добавь следующие стили, используя изученные селекторы:
- Стилизуй основное меню
- Сделай вложенные списки другого цвета
- Добавь отступы между пунктами меню
- Используй группировку селекторов, где это уместно
- Стилизуй ссылки на PDF-файлы
- Добавь иконку для внешних ссылок (с атрибутом target="_blank")
- Особо выдели ссылки с data-атрибутами

<details>
  <summary>Подсказка</summary>

  ```css
  /* Стили для основного меню */
  .main-menu {
    background-color: #f8f9fa;
    padding: 20px;
  }

  /* Убираем маркеры у всех списков внутри меню */
  ul {
    list-style: none;
    padding: 0;
    margin: 0;
  }

  /* Стили для всех пунктов меню */
  li {
    padding: 10px;
    margin: 5px 0;
    background-color: #ffffff;
  }

  /* Стили только для вложенных списков */
  > ul > li > ul {
    margin-left: 20px;
    margin-top: 10px;
  }

  /* Стили для пунктов вложенных списков */
  > ul > li > ul > li {
    background-color: #e9ecef;
  }

  /* Группировка селекторов для общих стилей */
  ul, li {
    border-radius: 5px;
  }

  /* Базовые стили для ссылок */
  .nav-link {
    text-decoration: none;
    color: #333;
  }

  /* Стили для PDF-ссылок */
  a[href$=".pdf"] {
    color: #dc3545;
    font-weight: bold;
  }

  a[href$=".pdf"]::after {
    content: " (PDF)";
    font-size: 0.8em;
  }

  /* Стили для внешних ссылок */
  a[target="_blank"]::after {
    content: " ↗";
    color: #6c757d;
  }

  /* Стили для ссылок с data-атрибутами */
  a[data-type="contact"] {
    background-color: #28a745;
    color: white;
    padding: 5px 10px;
    border-radius: 3px;
  }

  /* Стили для ссылок, начинающихся с https:// */
  a[href^="https://"] {
    color: #007bff;
  }
  ```
</details>
