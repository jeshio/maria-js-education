# 2.1 Введение в CSS

CSS (Cascading Style Sheets, каскадные таблицы стилей) — это язык, который отвечает за внешний вид веб-страниц. Если HTML определяет структуру и содержание, то CSS задаёт цвета, размеры, расположение и другие визуальные аспекты.

:::tip[Аналогия]
Если представить веб-страницу как дом, то HTML — это фундамент и каркас, JavaScript — это электрика и коммуникации, а CSS — это отделка, цвет стен, мебель и декор.
:::

## Для чего нужен CSS?

CSS позволяет:
- Изменять цвета текста и фона
- Задавать размеры элементов
- Определять расположение элементов на странице
- Создавать анимации и эффекты
- Адаптировать дизайн сайта под разные устройства

Без CSS все сайты выглядели бы как простой текст с базовым форматированием.

## Способы подключения CSS

Существует три способа добавить CSS к HTML:

### 1. Встроенные стили (inline)

Стили задаются прямо в HTML-теге с помощью атрибута `style`:

```html
<p style="color: red; font-size: 20px;">Этот текст будет красным и увеличенным.</p>
```

**Плюсы**: быстро и просто применить стиль к конкретному элементу.  
**Минусы**: стили смешиваются с HTML, трудно поддерживать, нельзя переиспользовать.

### 2. Внутренние стили (internal)

Стили размещаются в теге `<style>` в разделе `<head>` HTML-документа:

```html
<head>
  <style>
    p {
      color: blue;
      font-size: 16px;
    }
    .important {
      font-weight: bold;
    }
  </style>
</head>
<body>
  <p>Этот текст будет синим.</p>
  <p class="important">Этот текст будет синим и жирным.</p>
</body>
```

**Плюсы**: стили отделены от HTML, могут применяться к нескольким элементам.  
**Минусы**: не подходит для многостраничных сайтов (придётся дублировать код).

### 3. Внешний CSS-файл (external)

Стили выносятся в отдельный файл с расширением `.css`, который подключается к HTML с помощью тега `<link>`:

```html
<head>
  <link rel="stylesheet" href="styles.css">
</head>
```

Содержимое файла `styles.css`:
```css
p {
  color: green;
  font-size: 18px;
}
.important {
  font-weight: bold;
}
```

**Плюсы**: полное разделение содержания и оформления, переиспользование стилей на нескольких страницах, кеширование браузером.  
**Минусы**: требуется дополнительный запрос к серверу.

:::danger[Важно]
В реальных проектах почти всегда используется третий способ — внешние CSS-файлы. Это лучшая практика для разделения содержания и оформления.
:::

## Синтаксис CSS

Правило CSS состоит из:
1. **Селектора** — указывает, к каким HTML-элементам применяются стили
2. **Блока объявлений** — содержит одно или несколько объявлений, заключённых в фигурные скобки `{}`
3. **Объявлений** — пары "свойство: значение", которые разделяются точкой с запятой `;`

```css
селектор {
  свойство1: значение1;
  свойство2: значение2;
  /* и так далее */
}
```

Пример:
```css
p {
  color: blue;
  font-size: 16px;
  line-height: 1.5;
}
```

В этом примере:
- `p` — селектор (выбирает все параграфы)
- `color`, `font-size`, `line-height` — свойства
- `blue`, `16px`, `1.5` — значения свойств

## Базовые селекторы

CSS использует селекторы для определения, к каким элементам применять стили:

### Селектор по тегу

Применяет стиль ко всем элементам с указанным тегом:

```css
h1 {
  color: purple;
}
```

### Селектор по классу

Классы позволяют применять стили к группе элементов. Селектор класса начинается с точки:

```css
.important {
  font-weight: bold;
}
```

В HTML класс задаётся через атрибут `class`:
```html
<p class="important">Этот текст будет жирным.</p>
```

### Селектор по идентификатору

ID должен быть уникальным на странице. Селектор ID начинается с символа `#`:

```css
#header {
  background-color: black;
  color: white;
}
```

В HTML ID задаётся через атрибут `id`:
```html
<div id="header">Шапка сайта</div>
```

### Универсальный селектор

Звёздочка `*` выбирает все элементы на странице:

```css
* {
  margin: 0;
  padding: 0;
}
```

:::warning[Осторожно]
Универсальный селектор может замедлить работу страницы, если на ней много элементов. Используйте его осмотрительно.
:::

## Комментарии в CSS

В CSS комментарии начинаются с `/*` и заканчиваются `*/`:

```css
/* Это комментарий в CSS */
p {
  color: green; /* Это тоже комментарий */
}
```

## Практическое задание

Создай HTML-страницу с разными элементами и примени к ним стили всеми тремя способами:

1. Создай файл `index.html` со следующей структурой:
```html
<!DOCTYPE html>
<html>
<head>
  <title>Изучаем CSS</title>
  <!-- Здесь будут внутренние стили и ссылка на внешний CSS -->
</head>
<body>
  <h1>Основы CSS</h1>
  <p>Это обычный параграф.</p>
  <p class="important">Это важный параграф.</p>
  <p id="special">Это особенный параграф.</p>
  <div class="box">Это блок с классом "box".</div>
</body>
</html>
```

2. Добавь встроенные стили к заголовку (`color: purple; text-align: center;`)

3. Добавь внутренние стили в раздел `<head>`:
```html
<style>
  /* 1. Добавь стили к параграфу */
  /* font-size: 18px; */
  /* line-height: 1.5; */
  /* 2. Добавь стили к классу important */
  /* font-weight: bold; */
  /* color: red; */
</style>
```

4. Создай файл `styles.css`, это будут внешние стили:
```css
/* 3. Добавь стили к идентификатору special */
/* background-color: yellow; */
/* padding: 10px; */
/* border: 1px solid black; */

/* 4. Добавь стили к классу box */
/* width: 200px; */
/* height: 100px; */
/* background-color: lightblue; */
/* margin-top: 20px; */
/* text-align: center; */
/* padding-top: 40px; */
```

5. Подключи внешний CSS-файл в `<head>`:
```html
<link rel="stylesheet" href="styles.css">
```

6. Открой HTML-файл в браузере и посмотри, как применились разные стили.

7. Попробуй изменить стили в CSS и посмотри, как они применяются в браузере.
