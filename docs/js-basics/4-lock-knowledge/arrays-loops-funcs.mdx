## Задача 1: Сумма чётных чисел

:::info[Напоминание]

Чтобы проверить, является ли число чётным, можно использовать оператор `%` (остаток от деления):
```javascript
if (number % 2 === 0) {
  console.log('Чётное');
}
```
Чётные числа делятся на 2 без остатка.
:::

---

### Условие

Создай массив `numbers` с числами `1, 2, 3, 4, 5, 6`. Затем напиши функцию `getEvenSum`, которая принимает массив чисел, находит все чётные числа и возвращает их сумму.

Пример:
```javascript
console.log(getEvenSum([1, 2, 3, 4, 5, 6])); // Ожидаем: 12
```

    checks={[

    let sum = 0;
    for (let i = 0; i < numbers.length; i++) {

## Задача 2: Обратный порядок

:::tip

Чтобы перебирать массив в обратном порядке, можно начать с последнего элемента и двигаться к первому:
```javascript
for (let i = array.length - 1; i >= 0; i--) {
  console.log(array[i]);
}
```
Попробуй догадаться, почему `i` инициализируется со значением `array.length - 1`?
:::

---

### Условие

Создай функцию `reverseArray`, которая принимает массив и возвращает новый массив, элементы которого расположены в обратном порядке.

Пример:
```javascript
console.log(reverseArray(['яблоко', 'банан', 'вишня'])); // Ожидаем: ['вишня', 'банан', 'яблоко']
```

<CodeEditorWithTests
    checks={[
        { type: 'function', name: 'reverseArray', tests: [{ input: [['яблоко', 'банан', 'вишня']], output: ['вишня', 'банан', 'яблоко'] }] }

## Задача 3: Словарь

### Условие

Создай массив объектов `dictionary`, где каждый объект содержит два свойства:
1. `word` — слово на русском.
2. `translation` — его перевод на английский.

Пример:
```javascript
const dictionary = [
  { word: 'яблоко', translation: 'apple' },
  { word: 'банан', translation: 'banana' },
  { word: 'вишня', translation: 'cherry' },
];
```

Затем напиши функцию `translate`, которая принимает массив `dictionary` и слово на русском. Функция должна возвращать перевод слова на английский. Если слово не найдено, возвращай строку `"Перевод не найден"`.

Пример вызова:
```javascript
console.log(translate(dictionary, 'банан')); // Ожидаем: "banana"
console.log(translate(dictionary, 'груша')); // Ожидаем: "Перевод не найден"
```

    { word: 'банан', translation: 'banana' },
    { word: 'вишня', translation: 'cherry' },
];

function translate(dictionary, word) {
    for (let i = 0; i < dictionary.length; i++) {
        if (dictionary[i].word === word) {

}

console.log(translate(dictionary, 'груша')); // "Перевод не найден"
`}

## Задача 4: Работа с несколькими функциями

### Условие

Создай массив `numbers` с числами `10, 20, 30, 40, 50`. Напиши две функции:
1. `getAverage(numbers)` — возвращает среднее арифметическое чисел.
2. `getAboveAverage(numbers, average)` — возвращает массив чисел, которые больше среднего.

Пример:
```javascript
const numbers = [10, 20, 30, 40, 50];
const average = getAverage(numbers); // Ожидаем: 30
console.log(getAboveAverage(numbers, average)); // Ожидаем: [40, 50]
```

<LiveCodeEditor
    code={`const numbers = [10, 20, 30, 40, 50];

// Создай функцию getAverage

// Создай функцию getAboveAverage

// Проверь результат
`}
    solution={`const numbers = [10, 20, 30, 40, 50];

function getAverage(numbers) {
    let sum = 0;
    for (let i = 0; i < numbers.length; i++) {
        sum += numbers[i];
    }
    return sum / numbers.length;
}

function getAboveAverage(numbers, average) {
    let result = [];
    for (let i = 0; i < numbers.length; i++) {
        if (numbers[i] > average) {
            result.push(numbers[i]);
        }
    }
    return result;
}

const average = getAverage(numbers);
console.log(average); // 30
console.log(getAboveAverage(numbers, average)); // [40, 50]
`}
/>
