# 1.5 Булева логика

## Знания

:::tip[Что это такое?]
Происходит от фамилии английского математика (Джорджа Буля).

Булева логика — это система операций, работающая с двумя значениями: `true` (истина) и `false` (ложь). В программировании она используется для проверки условий и управления потоком выполнения программы.
:::

### **Основные операции**

#### 1. **Логическое И (`&&`)**
Возвращает `true`, если оба значения являются истиной.

**Пример:**
```javascript
let isStudent = true;
let hasID = true;

let canEnter = isStudent && hasID; // -> true (оба условия истинны)

let cannotEnter = isStudent && false; // -> false (одно из условий ложно)
```

#### 2. **Логическое ИЛИ (`||`)**
Возвращает `true`, если хотя бы одно значение является истиной.

**Пример:**
```javascript
let isAdmin = false;
let isModerator = true;

let canEdit = isAdmin || isModerator; // -> true (хотя бы одно условие истинно)
```

#### 3. **Логическое НЕ (`!`)**
Инвертирует значение. `true` становится `false`, и наоборот.

**Пример:**
```javascript
let isGuest = false;

let isNotGuest = !isGuest; // -> true (гость превращается в не-гостя)

// А можно даже так, если использовать скобки.
// Сначала вычисляется значение внутри скобок.
const isNotTest = !(isGuest || true) // -> false
```

:::tip[Про скобки и порядок операций]
Правило скобок действует как в математике. Сначала вычисляется то, что вунтри скобок, потом снаружи.

Также, сначала происходят операции умножения и деления, а потом операции суммирования и разности.

```javascript
const test = 2 + 2 * 2 / 2 - 2; // -> 2
const test2 = (2 + 2) * (2 / 2 - 2); // -> -4
```
:::

### **Сравнения**

Булева логика часто используется с операторами сравнения:
- `>` (больше)
- `<` (меньше)
- `>=` (больше или равно)
- `<=` (меньше или равно)
- `===` (строгое равенство)
- `!==` (строгое неравенство)

**Пример:**
```javascript
let age = 20;

let isAdult = age >= 18; // -> true (20 больше или равно 18)
let isTeen = age < 18; // -> false (20 не меньше 18)

let isExactMatch = age === 20; // -> true (значение совпадает с 20)
let isDifferent = age !== 15; // -> true (значение не совпадает с 15)
```

:::warning[Ещё раз про скобки]
- Используй скобки для уточнения приоритета операций, например: `(condition1 && condition2) || condition3`. _condition_ переводится как _условие_
:::

## Практическое задание

__Задача__: Выполни следующие шаги:

1. Объяви переменную `age` со значением `25`.
2. Объяви переменную `hasDrivingLicense` со значением `true`.
3. Объяви переменную `isEmployed` со значением `false`.
4. Создай переменную `canRentCar`, которая будет равна `true`, если `age` больше или равно `21` и у человека есть водительские права.
5. Создай переменную `isEligibleForLoan`, которая будет равна `true`, если человек старше `18` и либо трудоустроен, либо имеет водительские права.
6. Нажми Запустить и проверь, что всё правильно
7. Когда сделаешь всё правильно, то попробуй поизменять значения переменных и понажимать Запустить, чтобы увидеть как меняется результат. Делай это пока не поймешь всю логику - твоя цель, понять, __как одни значения влияют на другие__.
