# 1.6 Условия

## Знания

:::tip[Что это такое?]
Иногда нам нужно принимать решения в программе: если происходит одно, делаем это, а если другое — делаем совсем другое. Тут на сцену выходит **условный оператор**, который говорит: _"Давай проверим, что правда (true), а что нет (false)!"_.
:::

### **Основная форма `if`**

Синтаксис условия выглядит так:

```javascript
if (условие) { // если
  // Если условие истинно (true), выполнится этот код
} else { // иначе
  // А если условие ложно (false), выполнится этот код
}
```

### **Пример простого условия**

```javascript
let weather = 'солнечно';
let mood = ''

if (weather === 'солнечно') {
  mood = 'настроение ужасное'
} else {
  mood = 'настроение прекрасное'
}
```

### **Несколько условий**

Условие может состоять из нескольких элементов, каждый из которых будет проверяться.

```javascript
let isAdmin = false;
let isModerator = true;
let message = '';

if (isAdmin || isModerator) { // хотя бы одно из значений должно быть true
  message = 'У вас расширеные права пользователя';
} else {
  message = 'У вас обычные права пользователя';
}
```

### **Несколько проверок**

Иногда нужно проверить несколько условий подряд. Для этого можно использовать `else if`:

```javascript
const time = 10;
let message = '';

if (time < 12) { // если
  message = 'Доброе утро!';
} else if (time < 18) { // иначе если
  message = 'Добрый день!';
} else { // иначе
  message = 'Добрый вечер!';
}
```

В этом примере мы говорим браузеру:

\- Дорогой друг, напиши, пожалуйста, `Доброе утро!`, если время ещё до 12

\- Иначе же, если время более 12-ти, но раньше 18-ти, то прошу написать `Добрый день!`

\- Ну а во всех остальных случаях предлагаю приветствовать фразой `Добрый вечер!`

\- Кстати, время я для тебя задал в константу `time`

:::tip[На заметку]
Как только одно условие выполняется, остальные уже не проверяются!
:::

Другое дело тут:

```javascript
let time = 10;
let message = '';

if (time < 12) {
  // даже если это значение будет установлено,
  // в следующей конструкции оно всё равно перезапишеться (как думаешь почему?)
  message = 'Доброе утро!';
}

if (time < 18) {
  message = 'Добрый день!';
} else {
  message = 'Добрый вечер!';
}
```

### **Тернарный оператор**

Иногда можно написать условие более коротким способом. Например, когда в зависимости от условия, переменной или константе присваивается значение.

В этом случае, вместо `if` можно (но не обязательно) использовать **тернарный оператор** (_"тройной"_ от французского) — он компактнее, но делает то же самое:

```javascript
let age = 18;

// используем тернарный оператор
let canVote = age >= 18 ? 'Можно голосовать' : 'Еще рано голосовать';

// и то же самое с помощью if
if (age >= 18) {
    canVote = 'Можно голосовать'
}
else {
    canVote = 'Еще рано голосовать'
}
```

Как видишь, вместо 6-ти строчек всего одна. В общем вот синтаксис тернарного оператора:

```javascript
const primer = условие ? значие_если_условие_истино : значение_если_условие_ложно
```

2. Объяви переменную `message`
3. Напиши условие:
- Если `timeOfDay` больше `4` или меньше `12`, присвой переменной `message` значение `'Доброе утро!'`.

5. Нажми _Запустить_ и проверь своё решение
