# 1.4 Операции между переменными и константами

## Знания

:::tip[Что это такое?]
Операции позволяют выполнять различные действия с данными, хранящимися в переменных и константах. В JavaScript существуют различные типы операций для разных типов данных.
:::

### **Операции со строками (`string`)**

Строки можно объединять (___конкатенировать___) и выполнять над ними другие операции.

**Пример операций со строками:**

```javascript
let greeting = 'Привет, ';
const name = 'Маша';

let message = greeting + name; // Конкатенация строк -> 'Привет, Маша'

let upperCaseName = name.toUpperCase(); // Преобразование строки в верхний регистр -> 'МАША'

const messageWithName = message + '; сообщение от ' + upperCaseName // -> Привет, Маша; сообщение от МАША
```

### **Операции с числами (`number`)**

С числами можно выполнять математические операции: сложение, вычитание, умножение, деление и другие.

**Пример операций с числами:**

```javascript
let a = 10;
const b = 5;

let sum = a + b; // Сложение -> 15

let difference = a - b; // Вычитание -> 5

let product = a * b; // Умножение -> 50

let quotient = a / b; // Деление -> 2

let remainder = a % b; // Остаток от деления -> 0

let anotherRemainder = 12 % b; // Остаток от деления -> 2
```

### Всё-таки, зачем нужны константы если есть переменные?

Константы используются, когда знаешь, что их значение по ходу кода __не должно меняться__, а переменные наоборот! Вот пример кода.

```javascript
const name = "Маша";
let work = "Дизайнер";
let salary = 60000;
let currentWork = name + " профессиональный " + work + " и её зарплата " + salary + " рублей";
// -> Маша профессиональный Дизайнер и её зарплата 60000 рублей

work = "Программист"
salary = 150000;
currentWork = name + " профессиональный " + work + " и её зарплата " + salary + " рублей";
// -> Маша профессиональный Программист и её зарплата 150000 рублей
```
