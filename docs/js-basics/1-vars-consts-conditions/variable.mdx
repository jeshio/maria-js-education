# 1.2 Переменные

## Знания

:::tip[Что это такое?]
___Переменная___ — это "контейнер", который может хранить значение. В отличие от констант, значения переменной можно изменять после объявления.
:::

В JavaScript для объявления переменной используется __ключевое слово__ `let`.

Пример создания переменной:

```javascript
let myVariable = 'Начальное значение'; // Объявили переменную

myVariable = 'Новое значение'; // Изменили её значение, и это нормально!
```

`myVariable` в этом примере - это название переменной, а с помощью знака равно этой переменной присваивается значение, которое потом можно использовать или изменять.

:::tip[Как называть переменные и константы?]
Переменные и константы можно называть как угодно, но обязатнльно на латинице.

Среди разработчиков принято использовать английский язык для именования.

Поэтому я сразу в примерах использую названия на английском, нужно к этому привыкнуть и когда ты будешь думать как назвать что-то в коде, используй английский язык.
:::

:::warning[Особенности переменных]
- Переменные создаются с помощью let.
- Их значения можно менять.
:::

:::danger[Необъявленная переменная?]
Если вы попытаетесь использовать переменную, которую не объявили, JavaScript выдаст ошибку: `ReferenceError: <название переменной> is not defined.`

Это же относится и к константам, если помнишь.
:::
