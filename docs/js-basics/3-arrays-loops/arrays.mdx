# 3.1 Массивы

## Знания

:::tip[Что такое массивы?]
Массив — это структура данных, которая позволяет хранить несколько значений в одной переменной. Значения внутри массива могут быть любого типа: числа, строки, объекты, другие массивы и даже функции. Массивы используются для упрощения работы с большим количеством данных.
:::

### **Как выглядит массив?**

Массив создается с использованием квадратных скобок `[ ]`, а значения внутри разделяются запятыми:

```javascript
const fruits = ['яблоко', 'банан', 'вишня'];
console.log(fruits);
```

### **Как работать с массивами?**

1. **Доступ к элементам массива:**

Каждое значение в массиве имеет индекс — его порядковый номер, начиная с 0.

```javascript
const fruits = ['яблоко', 'банан', 'вишня'];
console.log(fruits[0]); // -> 'яблоко'
console.log(fruits[1]); // -> 'банан'
console.log(fruits[2]); // -> 'вишня'
```

2. **Добавление элементов:**

```javascript
const fruits = ['яблоко', 'банан'];
fruits.push('вишня'); // Добавляем новый элемент в конец массива
console.log(fruits); // -> ['яблоко', 'банан', 'вишня']
```

3. **Удаление элементов:**

```javascript
const fruits = ['яблоко', 'банан', 'вишня'];
fruits.pop(); // Удаляем последний элемент
console.log(fruits); // -> ['яблоко', 'банан']
```

4. **Изменение элемента:**

```javascript
const fruits = ['яблоко', 'банан', 'вишня'];
fruits[1] = 'апельсин'; // Заменяем второй элемент
console.log(fruits); // -> ['яблоко', 'апельсин', 'вишня']
```

5. **Длина массива:**

Чтобы узнать количество элементов в массиве, используйте свойство `length`.

```javascript
const fruits = ['яблоко', 'банан', 'вишня'];
console.log(fruits.length); // -> 3
```

:::tip
С массивами та же ситуация, как и с объектами - если для объявления массива использовался `const`, то элементы массива можно всё равно изменить, удалить или добавить.

Но всё так же мы не можем изменить саму константу:
```javascript
const cars = ['Hyundai','Lada']
cars = ['Toyota', 'Lada']; // <-- саму константу изменять нельзя, будет ОШИБКА
cars[0] = 'Toyota'; // <-- а это нормально, так делать МОЖНО
// В этом случае как раз и получится ['Toyota', 'Lada']
```
:::

## Редактор кода

Попробуй изменить порядок строк в коде так, чтобы в массиве `fruits` стало следующее значение:

```javascript
['яблоко', 'киви', 'груша']
```

Правило:
- Ты можешь только **переставлять строки местами**, но не добавлять, не изменять и не удалять код.

:::tip
В редакторах кода можно удобно переставить всю строку. Для этого:
1. Поставь __каретку курсора__ (это когда ты редактируешь текст и у тебя мигает палочка, где ты сейчас его вводишь) в любое место строки, но ничего не выделяй
2. Нажми Command+X - вся строка вырежтся из кода
3. Поставь каретку курсора на строку, __сверху которой__ хочешь вставить вырезаную строку
4. Нажми Command+V

С первого раза может не получится, но это легко запомнить, когда попробуешь несколько раз. Переставлять строки нужно довольно часто, поэтому рекомендую запомнить этот быстрый способ.
:::
