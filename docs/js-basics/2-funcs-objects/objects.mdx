# 2.2 Объекты: свойства и методы

## Знания

:::tip[Что такое объект?]
Объект — это "контейнер", который позволяет хранить данные и действия в одном месте. Если переменные и функции — это отдельные инструменты, то объект их объединяет. Он может содержать свойства (данные) и методы (действия).
:::

### **Объект — это супер-переменная**

Мы уже знаем, что переменная хранит значение. Объект может хранить не одно, а сразу много значений в виде свойств. Причем эти свойства можно изменять, добавлять новые, а также удалять ненужные.

```javascript
const dog = {
  name: 'Того',
  age: 4,
  isHungry: true
};
```

- `dog` - это объект
- `name`, `age` и `isHungry` — это свойства объекта.
- `Того`, `4` и `true` — их значения.

### **Методы — это функции внутри объекта**

Мы знаем, что функции помогают повторно использовать код. В объекте функции называются методами, а их главным преимуществом является то, что они могут работать с данными этого же объекта (в примере этого пока не видно).

```javascript
const dog = {
  name: 'Того',
  bark: function() {
    return 'Гав-гав!';
  }
};

const result = dog.bark(); // -> 'Гав-гав!'
```

Метод `bark` — это обычная функция, но теперь она является частью объекта `dog`.

### **Зачем нам объекты?**

Представь, что у тебя есть собака (хотя в твоем случае — это уже реальность). Вместо того чтобы заводить кучу переменных для описания Того (его имя, возраст, голоден он или нет), можно объединить всё в одном объекте.

```javascript
const dog = {
  name: 'Того',
  age: 4,
  isHungry: true
};
```

Теперь ты можешь легко получать доступ к данным:

```javascript
dog.name; // -> 'Того'
const howManyYears = dog.age; // -> 4
```

Или изменять их:

```javascript
dog.isHungry = false;
const isTogoHungry = dog.isHungry; // -> false
```

:::tip[Как объекты делают жизнь проще?]
- **Собранность данных:** Всё, что связано с одним "понятием" (например, собака), хранится в одном месте.
- **Меньше переменных:** Вместо множества отдельных переменных для описания чего-то (например, имени, возраста и цвета) достаточно одного объекта.
- **Читаемость:** Код становится логичнее, когда данные сгруппированы.
:::

__Задача__: Создай объект `book` и поработай с его свойствами и методами:

1. Создай объект `book` со следующими свойствами:
- `title` со значением `'Доводы рассудка'`.
- `author` со значением `Джейн Остин`.
- `mainCharacter` со значением `Энн`.
2. Добавь свойство `year` со значением `1818`.
3. Добавь метод `read`, который возвращает строку `'Здесь как-бы текст книги'`.
4. Используя свойства объекта создай переменную `whatWeKnowAboutBook`. В результате должна получится строка:
`'Книга Доводы рассудка написана в 1818 году, автор Джейн Остин'`
