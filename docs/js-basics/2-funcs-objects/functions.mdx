# 2.1 Функции

## Знания

:::tip[Что такое функции?]
Функции — это основной инструмент для организации и многократного использования одного и того&nbsp;же кода, где-бы он не пригодился. С помощью функций можно "упаковать" набор инструкций, а затем вызывать их в нужное время. Это помогает сократить дублирование кода и делает программы более структурированными.
:::

### **Как создается функция?**

Всё очень просто. Функция создается с помощью ключевого слова `function`, за которым следует имя функции, круглые скобки `( )` и фигурные скобки `{ }`, содержащие инструкции.

```javascript
function имяФункцииПравдаЛатиницей() {
  // Здесь код, который выполнит функция
}
```

### **Пример создания и вызова функции**

```javascript
let message = ''

function sayHello() {
    message = 'Hello, world!'
}

// А тут вызов функции. В результате переменная message будет содержать строку 'Hello, World!'
sayHello();
```

Когда вызываешь функцию, как в примере выше, программа выполняет код внутри неё.

### **Функции с параметрами**

Магия функций в том, что в функцию можно передавать данные через `параметры`. Параметры записываются в круглых скобках:

```javascript
let message = ''

function greet(name) {
  message = 'Приветик, ' + name + '!';
}

greet('Маша'); // В результате вызова переменная message будет содержать 'Привет, Маша!'
greet('Того'); // А теперь - 'Привет, Того!'
```

### **Функции с возвращаемым значением**

Функции могут не просто выполнять инструкции, но и возвращать результат своей работы с помощью ключевого слова `return`.

```javascript
function sum(a, b) {
  return a + b;
}

let result = sum(5, 10); // Функция вернет 15, и это значение будет сохранено в переменной result
result = sum(2, 3); // А теперь в result будет находится число 5
```

:::tip[Почему функции важны?]
- __Функции помогают избежать дублирования кода__ - тебе не придется писать одно и то же в разных частях своей программы.
- __Они позволяют логически разделить код на части__ - можно создать функцию, которая приветствует, или решает какую-то сложную формулу, или делает что-либо, что нужно делать много раз.
- __Функции легко тестировать__ - всего-лишь поменяй параметры, которые ты передаешь в неё и уже другой результат.
:::
