# 2.3 Консоль

## Знания

:::tip[Существующие функции]
В позапрошлой главе ты писала свои функции, это очень важно. Но в JavaScript уже есть большое количество готовых функций.

Кроме того, тысячи программистов по всему миру ежедневно публикуют какие-то свои функции в Интернет. В будущем ты научишься их скачивать очень удобным способом. Всё уже давно продумано.
:::

### **Функция `console.log`**

Начнём мы с первой важной функции, которая всегда пригодится. С помощью неё ты можешь посмотреть, например, какие значения содержаться в переменных на определенном этапе кода. К примеру, внутри условия до того как ты присвоешь новое значение переменной.

Это бывает нужно когда код большой, и сходу не совсем понятна его работа или когда-ты написала большую функцию и хочешь посмотреть как хорошо она отрабатывает, либо ты видишь какую-то ошибку, но не понятно почему она происходит и тогда везде расставляешь `console.log`, чтобы проследить как изменеются твои переменные. Этот процесс поиска, кстати, называется __отладка__.

Как видишь способов применения у этой функции много, а теперь посмотри как она выглядит:


**Пример использования `console.log`:**

```javascript
let message = 'Привет, мир!';
console.log(message); // Выведет в консоль: Привет, мир!
```

### Что такое консоль?

__Консоль__ - это спецаильное пространство, где ты можешь выполнять команды с помощью того же JS, чтобы, например, протестировать их перед тем, как писать код. Но чаще консоль используется для просмотра JavaScript ошибок на странице и выводов console.log.

Когда ты открываешь любую страницу в браузере на компьютере, то ты можешь откурыть консоль для неё. Иногда там может что-то быть. Но считается правильным, чтобы консоль использовалась только при разработке, а уже рабочий сайт для пользователей ничего не выводит в консоль. Но это не всегда получается.

Вот как открыть консоль:

2. Нажми Инспектировать

4. Готово! Можно вводить команды, но пока мы не будем.

### **Другие полезные функции**

JavaScript предоставляет множество встроенных функций, которые облегчают работу с данными и выполняют различные задачи.

- `alert()`: Показывает всплывающее окно с сообщением.
- `prompt()`: Запрашивает ввод пользователя.
- `Math.random()`: Генерирует случайное число.
- `Date()`: Работает с датами и временем.

**Пример использования `Math.random()`:**

```javascript
// В randomNumber попадёт случайное число между 0 и 1, например 0.3851107028443719
let randomNumber = Math.random();
```

## Редактор кода
Для работы с кодом программисты используют редактор кода. Ты уже писала код в симуляторе, который я сделал. Но вообще, этот симулятор не совсем полноценный. Поэтому с этого момента ты будешь работать также и с полноценным редактором кода, правда пока всё ещё в браузере, а не в отдельной программе.

Кокретно сейчас отличие от прошлого редактора в том, что в этом редакторе есть консоль, куда ты можешь что-то выводить. В будущих заданиях там можно будет выводить и HTML, и CSS, и смотреть сразу же как это выглядит.

К сожалению, этот редактор не позволить проверить твой код, поэтому к моей версии мы всё равно будем возвращаться. А этот новый редактор - площадка для игр - реально, так называют подобные штуки - `playground`. Кстати код, здесь запускается автоматически когда ты его изменяешь, поэтому нет кнопки `Запустить`.

_Чтобы этот редактор работал возможно понадобиться включить VPN._

В этом примере уже введён код для ознакомления:

:::warning[Ошибки в консоле]
В консоле также показываются ошибки в твоем коде.
Они могут показываться и пока ты не дописала код по мере ввода, поэтому __сначала допиши__.

А вообще текст ошибки может помочь тебе разобраться в чём проблема.
:::

## Практическое задание

__Задача__: Используй функции `console.log`, `Math.random` и собственные функции для выполнения следующих действий:

1. Объяви переменную `greeting` со значением `'Здравствуйте!'` и выведи ее в консоль.
2. Создай функцию `generateRandomNumber`, которая возвращает случайное число от 0 до 100, и выведи результат вызова этой функции в консоль.
    - `Math.random() * 100` - случайное число от 0 до 100
3. Измени значение переменной `greeting` на `'Привет!'` и снова выведи в консоль
4. Создай функцию `sayGoodbye`, которая выводит в консоль (НЕ возвращает) сообщение `'До свидания!'`.
5. Вызови функцию `sayGoodbye`.

Если всё правильно в консоле должно отобразиться следующее:
```
Здравствуйте!
Привет!
14.451967253757347 // тут будет твое случаное число
До свидвания!
```
